# 搭建虚拟环境
## 创建虚拟环境
- uv python install 3.11
- uv venv --python 3.11
## 激活虚拟环境
PowerShell中
```powershell
.\.venv\Scripts\Activate.ps1
```
cmd中
```cmd
.venv\Scripts\activate.bat
```
## (配置国内镜像)
1. 命令行一次性
uv pip install langchain --index-url https://mirrors.aliyun.com/pypi/simple
2. 文件配置
用户目录或项目根目录下 创建uv.toml文件,uv会自动读取
```
[[index]]
url = "https://mirrors.aliyun.com/pypi/simple/"
default = true
```
